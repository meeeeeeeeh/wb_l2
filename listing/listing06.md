Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод:
[3 2 3]

Слайс представляет из себя структуру, состоящую из указателя на массив данных,
длины и емкости. Если в слайс добавляется новый элемент, но его емкость уже исчерпана
то происходит перевыделение памяти - создается новая структура с большим объемом памяти и 
все элементы массива копируются в ее массив. 
При этом изначальным слайс не изменяется и если его не переприсвоить новой структуре то
добавления элемента не произойдет.

В примере слайс передается как копия. Изменеие первого элемента мы увидим в выводе, 
так как изменяется слайс, который был передан, но так как дальше происходит добавление 
элемента (соответствено выделение памяти и создание новой структуры), 
то все дальнейшие изменения происходят не с переданным слайсом, а с новым.
Так как созданный в функции слайс не ретернится в мейн, изменений с ним в выводе мы не увидим.
```
