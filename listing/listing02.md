Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Ключевое слово defer используется для откладывания выполнения функций.
То что стоит после defer будет выполнено после всех остальных функций и выаолнения return,
но перед выходом из функции

вывод:
2
1

В первом случае мы сначала доходим до return и ничего не возвращаем,
а потом увеличиваем х. Таким образом после окончания работы функции х равен 2
Во втором случае сначала будет выполнен return (функция вернет значение 1), 
а затем увеличивает его. Поэтому возвращаемое значение будет 1

```
